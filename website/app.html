Reviews

Feedback on quality, maintainability, and security for pull requests.

Apply suggestions

Actionable code suggestions can be applied back to a PR branch when authoring is enabled and the person invoking the command has write or admin access.

Commands

Use /analyze for a fresh review, /apply for suggestions, and /pause, /resume, or /status for auto analysis.

Command replies stay brief: 🙂 for normal replies, 😅 when something needs attention, and 😴 for paused or no-op states.
